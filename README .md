
# CONTACT MANAGEMENT SYSTEM

A simple Contact Management System built using C++.\
This is a beginner-level project created to practice basic C++ concepts and logic building.
## Features
* Display all contact details
* Add a new contact
* Search for a contact by name
* Edit contact details
* Delete a contact

## Information Stored for Each Contact
Each contact stores the following information:

* Name
* Contact Number
* Address
* Date of Birth (DOB)
* College Name
## Concepts Used
This project uses the following C++ concepts:

* Classes and Objects
* Vectors
* Strings
* Functions
* Sorting
* Binary Search
* Loops and Conditional Statements
## Purpose of the Project
The main goal of this project is to improve problem-solving skills and understand how data can be stored and managed using C++.
